Aula de Revisão: Letras do Hiragana

Por: Augusto, 09 de outubro de 2023.

Bem-vindo à nossa aula de revisão das letras do hiragana! O hiragana é um dos sistemas de escrita japonesa e é fundamental para a comunicação escrita em japonês. Vamos revisar todas as letras do hiragana e suas respectivas pronúncias.


Grupo das Vogais


Primeiro, vamos começar com o grupo das vogais. São cinco letras no grupo das vogais:


あ (a) い (i) う (u) え (e) お (o)

Pratique pronunciando cada uma delas. Elas são a base para muitas outras letras do hiragana.


Grupos das Consoantes


Agora, vamos começar com o grupo das vogais. São cinco letras no grupo das vogais:


か (ka) き (ki) く (ku) け (ke) こ (ko)

Essas letras representam sons como "ka", "ki", "ku", "ke" e "ko". Pratique pronunciar cada uma delas para se familiarizar com os sons.


Grupo das Consoantes com "S"


Agora, vejamos o grupo das consoantes com "s". Essas letras têm um som "s" acrescentado à consoante:


さ (sa) し (shi) す (su) せ (se) そ (so)

Pratique pronunciar essas letras e preste atenção ao som "s" adicionado.


Grupo das Consoantes com "T"


Agora, vamos para o grupo das consoantes com "t". Essas letras têm um som "t" acrescentado à consoante:


た (ta) ち (chi) つ (tsu) て (te) と (to)

Pratique pronunciar essas letras e preste atenção ao som "t" adicionado.


Grupo das Consoantes com "H"


Por fim, temos o grupo das consoantes com "h". Essas letras têm um som "h" acrescentado à consoante:


は (ha) ひ (hi) ふ (fu) へ (he) ほ (ho)

Pratique pronunciar essas letras e preste atenção ao som "h" adicionado.


Importância dos Flash Cards


Uma ferramenta útil para praticar e memorizar o hiragana são os flash cards. Eles são cartões com letras do hiragana de um lado e suas pronúncias do outro. Você pode usar flash cards físicos ou optar por recursos online, como o Quizlet, que oferece uma coleção de flash cards de hiragana disponíveis no seguinte link:

Os flash cards são uma maneira eficaz de testar seu conhecimento e aprimorar sua memória das letras do hiragana. Pratique regularmente com eles para fortalecer suas habilidades de leitura e escrita.


Continue praticando e explorando a beleza do hiragana, e em breve você estará mais confiante na escrita e leitura em japonês!
